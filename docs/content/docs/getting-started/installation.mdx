# Installation

Install the Integrate SDK using your preferred package manager.

## Using Bun (Recommended)

```bash
bun add integrate-sdk
```

## Using npm

```bash
npm install integrate-sdk
```

## Using pnpm

```bash
pnpm add integrate-sdk
```

## Using Yarn

```bash
yarn add integrate-sdk
```

## TypeScript Support

The SDK is built with TypeScript and includes type definitions out of the box. No additional `@types` packages are needed.

## Requirements

- Node.js 16+ or Bun 1.0+
- TypeScript 4.5+ (for TypeScript projects)

## Verify Installation

After installation, verify the SDK is working by importing it:

```typescript
import { createMCPClient } from "integrate-sdk";

console.log("Integrate SDK installed successfully!");
```

## Next Steps

Now that you have the SDK installed, check out the [Quick Start](/docs/getting-started/quick-start) guide to create your first integration.
