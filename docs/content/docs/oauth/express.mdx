## Installation

Install the Integrate SDK in your Express project:

```bash
bun add integrate-sdk
```

## Mount the Handler

Add OAuth route handlers to your Express application:

```typescript
import express from 'express';
import { toExpressHandler } from 'integrate-sdk/adapters/express';

const app = express();
app.use(express.json());

const handler = toExpressHandler({
  providers: {
    github: {
      clientId: process.env.GITHUB_CLIENT_ID!,
      clientSecret: process.env.GITHUB_CLIENT_SECRET!,
      redirectUri: 'http://localhost:3000/api/integrate/oauth/callback',
    },
  },
});

// Register OAuth routes
app.post('/api/integrate/oauth/authorize', handler.authorize);
app.post('/api/integrate/oauth/callback', handler.callback);
app.get('/api/integrate/oauth/status', handler.status);
app.post('/api/integrate/oauth/disconnect', handler.disconnect);

app.listen(3000, () => {
  console.log('Server running on port 3000');
});
```

## Configuration

Add your OAuth credentials to your `.env` file:

```bash
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret
```

## CORS Setup

If your frontend is on a different origin, configure CORS:

```typescript
import cors from 'cors';

app.use(cors({
  origin: 'http://localhost:3001',
  credentials: true,
}));
```

## Usage Examples

### Client-Side Authorization

```typescript
import { createMCPClient, githubPlugin } from 'integrate-sdk';

const client = createMCPClient({
  plugins: [
    githubPlugin({
      scopes: ['repo', 'user'],
    }),
  ],
});

// Start OAuth flow
await client.authorize('github');
```

## Framework-Specific Notes

- **Body Parser**: Use `express.json()` middleware to parse JSON request bodies.
- **CORS**: Install `cors` package for cross-origin requests: `bun add cors`.
- **Environment Variables**: Use `dotenv` package or set them directly.
