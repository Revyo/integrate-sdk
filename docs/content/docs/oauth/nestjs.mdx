## Installation

Install the Integrate SDK in your NestJS project:

```bash
bun add integrate-sdk
```

## Setup

### 1. Server Configuration

Create a server configuration file with your OAuth credentials:

```typescript
// lib/integrate-server.ts
import { createMCPServer, githubPlugin, gmailPlugin } from 'integrate-sdk/server';

export const { client: serverClient, handler } = createMCPServer({
  plugins: [
    githubPlugin({
      clientId: process.env.GITHUB_CLIENT_ID!,
      clientSecret: process.env.GITHUB_CLIENT_SECRET!,
      scopes: ['repo', 'user'],
    }),
    gmailPlugin({
      clientId: process.env.GMAIL_CLIENT_ID!,
      clientSecret: process.env.GMAIL_CLIENT_SECRET!,
      scopes: ['gmail.send', 'gmail.readonly'],
    }),
  ],
});
```

### 2. Mount the Handler

Create a controller to handle OAuth routes:

```typescript
// oauth.controller.ts
import { Controller, All, Req, Res } from '@nestjs/common';
import { Request, Response } from 'express';
import { handler } from '@/lib/integrate-server';
import { toNodeHandler } from 'integrate-sdk';

@Controller('api/auth')
export class OAuthController {
  private nodeHandler = toNodeHandler(handler);

  @All('*')
  handleOAuth(@Req() req: Request, @Res() res: Response) {
    return this.nodeHandler(req, res);
  }
}
```

Register the controller in your module:

```typescript
// app.module.ts
import { Module } from '@nestjs/common';
import { OAuthController } from './oauth.controller';

@Module({
  controllers: [OAuthController],
})
export class AppModule {}
```

## Configuration

Add your OAuth credentials to your `.env` file:

```bash
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret
```

## CORS Setup

Configure CORS in your `main.ts`:

```typescript
import { NestFactory } from '@nestjs/core';
import { AppModule } from './app.module';

async function bootstrap() {
  const app = await NestFactory.create(AppModule);
  
  app.enableCors({
    origin: 'http://localhost:3001',
    credentials: true,
  });

  await app.listen(3000);
}
bootstrap();
```

## Usage Examples

### Client-Side Authorization

```typescript
import { createMCPClient, githubPlugin } from 'integrate-sdk';

const client = createMCPClient({
  plugins: [
    githubPlugin({
      scopes: ['repo', 'user'],
    }),
  ],
});

// Start OAuth flow
await client.authorize('github');
```

## Framework-Specific Notes

- **Decorators**: NestJS uses decorators for routing and dependency injection.
- **Modules**: Organize your OAuth controller in a dedicated module for better structure.
- **Guards**: You can create custom guards to protect routes based on OAuth status.
