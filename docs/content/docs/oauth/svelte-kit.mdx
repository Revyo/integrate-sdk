## Installation

Install the Integrate SDK in your SvelteKit project:

```bash
bun add integrate-sdk
```

## Setup

### 1. Server Configuration

Create a server configuration file with your OAuth credentials:

```typescript
// lib/integrate-server.ts
import { createMCPServer, githubPlugin, gmailPlugin } from 'integrate-sdk/server';

export const { client: serverClient, handler } = createMCPServer({
  plugins: [
    githubPlugin({
      clientId: process.env.GITHUB_CLIENT_ID!,
      clientSecret: process.env.GITHUB_CLIENT_SECRET!,
      scopes: ['repo', 'user'],
    }),
    gmailPlugin({
      clientId: process.env.GMAIL_CLIENT_ID!,
      clientSecret: process.env.GMAIL_CLIENT_SECRET!,
      scopes: ['gmail.send', 'gmail.readonly'],
    }),
  ],
});
```

### 2. Mount the Handler

#### Option 1: Using Route Handlers

Create a catch-all API route at `src/routes/api/integrate/[...all]/+server.ts`:

```typescript
import { toSvelteKitHandler } from 'integrate-sdk/adapters/svelte-kit';
import { handler } from '$lib/integrate-server';

const svelteKitRoute = toSvelteKitHandler(handler);

export const POST = svelteKitRoute;
export const GET = svelteKitRoute;
```

#### Option 2: Using Hooks

Integrate OAuth handling into your `hooks.server.ts`:

```typescript
import { svelteKitHandler } from 'integrate-sdk/adapters/svelte-kit';
import { handler } from '$lib/integrate-server';

export async function handle({ event, resolve }) {
  return svelteKitHandler({ 
    authConfig: handler,
    event, 
    resolve,
    basePath: '/api/integrate'
  });
}
```

## Configuration

Add your OAuth credentials to your `.env` file:

```bash
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret
```

## Usage Examples

### Client-Side Authorization

Create a client configuration file:

```typescript
// lib/integrate.ts
import { createMCPClient, githubPlugin } from 'integrate-sdk';

export const client = createMCPClient({
  plugins: [
    githubPlugin({
      scopes: ['repo', 'user'],
    }),
  ],
  oauthFlow: { mode: 'popup' },
});
```

Then use it in your components:

```typescript
import { client } from '$lib/integrate';

// Start OAuth flow
await client.authorize('github');
```

## Framework-Specific Notes

- **Hooks**: Use the `svelteKitHandler` function to integrate OAuth into your hooks.
- **Environment Variables**: SvelteKit uses Vite for environment variables.
- **TypeScript**: Full TypeScript support with SvelteKit's type system.
