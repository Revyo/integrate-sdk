## Installation

Install the Integrate SDK in your Nitro project:

```bash
bun add integrate-sdk
```

## Mount the Handler

Create a catch-all API route in your Nitro project. Create a file at `routes/api/integrate/oauth/[...all].ts`:

```typescript
import { toNitroHandler } from 'integrate-sdk/adapters/nitro';

const handler = toNitroHandler({
  providers: {
    github: {
      clientId: process.env.GITHUB_CLIENT_ID!,
      clientSecret: process.env.GITHUB_CLIENT_SECRET!,
      redirectUri: 'http://localhost:3000/api/integrate/oauth/callback',
    },
  },
});

export default defineEventHandler(handler);
```

## Configuration

Add your OAuth credentials to your `.env` file:

```bash
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret
```

## Usage Examples

### Client-Side Authorization

```typescript
import { createMCPClient, githubPlugin } from 'integrate-sdk';

const client = createMCPClient({
  plugins: [
    githubPlugin({
      scopes: ['repo', 'user'],
    }),
  ],
});

// Start OAuth flow
await client.authorize('github');
```

## Framework-Specific Notes

- **Universal**: Nitro is a universal server framework that works across multiple platforms.
- **Event Handler**: Uses `defineEventHandler` for route definitions.
- **Deployment**: Can be deployed to various platforms (Vercel, Netlify, Cloudflare, etc.).
