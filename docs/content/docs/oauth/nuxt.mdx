## Installation

Install the Integrate SDK in your Nuxt project:

```bash
bun add integrate-sdk
```

## Mount the Handler

Create a catch-all API route in your Nuxt project. Create a file at `server/api/integrate/oauth/[...all].ts`:

```typescript
import { toNuxtHandler } from 'integrate-sdk/adapters/nuxt';

const handler = toNuxtHandler({
  providers: {
    github: {
      clientId: process.env.GITHUB_CLIENT_ID!,
      clientSecret: process.env.GITHUB_CLIENT_SECRET!,
      redirectUri: 'http://localhost:3000/api/integrate/oauth/callback',
    },
  },
});

export default defineEventHandler(handler);
```

## Configuration

Add your OAuth credentials to your `.env` file:

```bash
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret
```

## Usage Examples

### Client-Side Authorization

```typescript
import { createMCPClient, githubPlugin } from 'integrate-sdk';

const client = createMCPClient({
  plugins: [
    githubPlugin({
      scopes: ['repo', 'user'],
    }),
  ],
});

// Start OAuth flow
await client.authorize('github');
```

## Framework-Specific Notes

- **Event Handler**: Nuxt uses `defineEventHandler` for API routes.
- **Environment Variables**: Nuxt automatically loads `.env` files.
- **TypeScript**: Full TypeScript support with auto-imports.
