## Installation

Install the Integrate SDK in your Remix project:

```bash
bun add integrate-sdk
```

## Mount the Handler

Create separate route files for each OAuth action in your Remix application:

### Authorize Route

```typescript
// app/routes/api.integrate.oauth.authorize.ts
import { toRemixHandler } from 'integrate-sdk/adapters/remix';

const handler = toRemixHandler({
  providers: {
    github: {
      clientId: process.env.GITHUB_CLIENT_ID!,
      clientSecret: process.env.GITHUB_CLIENT_SECRET!,
      redirectUri: 'http://localhost:3000/api/integrate/oauth/callback',
    },
  },
});

export const action = handler.authorize;
```

### Callback Route

```typescript
// app/routes/api.integrate.oauth.callback.ts
import { toRemixHandler } from 'integrate-sdk/adapters/remix';

const handler = toRemixHandler({
```

## Configuration

Add your OAuth credentials to your `.env` file:

```bash
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret
```

## Usage Examples

### Client-Side Authorization

```typescript
import { createMCPClient, githubPlugin } from 'integrate-sdk';

const client = createMCPClient({
  plugins: [
    githubPlugin({
      scopes: ['repo', 'user'],
    }),
  ],
});

// Start OAuth flow
await client.authorize('github');
```

## Framework-Specific Notes

- **Loaders vs Actions**: Use `action` for POST requests (authorize, callback, disconnect) and `loader` for GET requests (status).
- **Environment Variables**: Remix loads `.env` files automatically. Access them via `process.env`.
- **TypeScript**: Remix has first-class TypeScript support.
