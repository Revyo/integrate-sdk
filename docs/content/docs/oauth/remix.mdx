## Installation

Install the Integrate SDK in your Remix project:

```bash
bun add integrate-sdk
```

## Mount the Handler

Create a catch-all route in your Remix application:

```typescript
// app/routes/api.auth.$.ts
import { OAuthHandler } from 'integrate-sdk';

const handler = new OAuthHandler({
  providers: {
    github: {
      clientId: process.env.GITHUB_CLIENT_ID!,
      clientSecret: process.env.GITHUB_CLIENT_SECRET!,
      redirectUri: 'http://localhost:3000/api/auth/callback',
    },
  },
});

export const action = async ({ request }: { request: Request }) => {
  return handler.handler(request);
};

export const loader = async ({ request }: { request: Request }) => {
  return handler.handler(request);
};
```

## Configuration

Add your OAuth credentials to your `.env` file:

```bash
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret
```

## Usage Examples

### Client-Side Authorization

```typescript
import { createMCPClient, githubPlugin } from 'integrate-sdk';

const client = createMCPClient({
  plugins: [
    githubPlugin({
      scopes: ['repo', 'user'],
    }),
  ],
});

// Start OAuth flow
await client.authorize('github');
```

## Framework-Specific Notes

- **Loaders vs Actions**: Use `action` for POST requests (authorize, callback, disconnect) and `loader` for GET requests (status).
- **Environment Variables**: Remix loads `.env` files automatically. Access them via `process.env`.
- **TypeScript**: Remix has first-class TypeScript support.
