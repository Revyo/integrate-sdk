## Installation

Install the Integrate SDK in your Astro project:

```bash
bun add integrate-sdk
```

## Setup

### 1. Server Configuration

Create a server configuration file with your OAuth credentials:

```typescript
// lib/integrate-server.ts
import {
  createMCPServer,
  githubPlugin,
  gmailPlugin,
} from "integrate-sdk/server";

export const { client: serverClient, handler } = createMCPServer({
  plugins: [
    githubPlugin({
      clientId: import.meta.env.GITHUB_CLIENT_ID!,
      clientSecret: import.meta.env.GITHUB_CLIENT_SECRET!,
      scopes: ["repo", "user"],
    }),
    gmailPlugin({
      clientId: import.meta.env.GMAIL_CLIENT_ID!,
      clientSecret: import.meta.env.GMAIL_CLIENT_SECRET!,
      scopes: ["gmail.send", "gmail.readonly"],
    }),
  ],
});
```

### 2. Mount the Handler

Create a catch-all API route at `pages/api/integrate/[...all].ts`:

```typescript
import { toAstroHandler } from "integrate-sdk/server";
import { handler } from "@/lib/integrate-server";
import type { APIRoute } from "astro";

export const ALL: APIRoute = toAstroHandler(handler, {
  redirectUrl: "/dashboard", // Where to redirect after successful OAuth
  errorRedirectUrl: "/auth-error", // Where to redirect on OAuth errors
});
```

## Configuration

Add your OAuth credentials to your `.env` file:

```bash
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret
```

## Usage Examples

### Client-Side Authorization

Create a client configuration file:

```typescript
// lib/integrate.ts
import { createMCPClient, githubPlugin } from "integrate-sdk";

export const client = createMCPClient({
  plugins: [
    githubPlugin({
      scopes: ["repo", "user"],
    }),
  ],
  oauthFlow: { mode: "popup" },
});
```

Then use it in your components:

```typescript
import { client } from "@/lib/integrate";

// Start OAuth flow
await client.authorize("github");
```

### Callback Handling

The OAuth callback is automatically handled by your catch-all route. After successful authorization, the access token will be available to your client.

### Status Checking

Check if a provider is authorized:

```typescript
const status = await client.checkAuthStatus("github");
console.log(status.authorized); // true/false
```

## Framework-Specific Notes

- **CORS**: If your Astro app serves from a different origin than your API routes, configure CORS in your `astro.config.mjs`.
- **Environment Variables**: Astro automatically loads `.env` files. Use `import.meta.env` for client-side variables.
- **TypeScript**: Astro has excellent TypeScript support out of the box.
